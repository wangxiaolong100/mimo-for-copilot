MiMo models should appear in the Copilot model picker as soon as the extension is active. If an API key is not configured yet, they show a warning icon until you run MiMo: Set API Key.

If you do not see them right away, the model list may simply be long. Scroll down in the picker and look for MiMo V2.5 Pro, MiMo V2.5 Omni, and other MiMo models.
